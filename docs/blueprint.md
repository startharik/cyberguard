# **App Name**: CyberMind

## Core Features:

- AI Chatbot: Cybersecurity Q&A: Engage with an AI chatbot to answer cybersecurity questions.
- User Dashboard: Personalized Dashboard: Welcome the user and present key options: start quiz, talk to AI, view quiz results summary.
- Interactive Quizzes: MCQ Quizzes: Offer multiple-choice quizzes on cybersecurity topics. Persist user profiles to a local JSON file.
- Results Display: Quiz Results Summary: Display a summary of the user's latest quiz results on the dashboard. Persist the latest quiz results to local storage.
- User Authentication: Login/Registration: Allow users to register and log in to access the platform. Persist user details to a local JSON file.
- Admin Quiz Management: Admin Interface: Provide an admin interface for managing and updating quizzes. Data is persisted to a JSON file.

## Style Guidelines:

- Primary color: Deep blue (#2E5984), evoking trust and security, without being a cliche.
- Background color: Light blue-gray (#E0E5EC), for a clean, secure-feeling backdrop.
- Accent color: Teal (#468975), used to draw attention to important elements. Use judiciously for a professional appearance.
- Body font: 'Inter', sans-serif. Headline font: 'Space Grotesk', sans-serif, to create some separation between body and header fonts.
- Use outline-style icons for a modern and clean aesthetic, with a bias towards the style of the heroicons library.
- Maintain a clean and structured layout, ensuring easy navigation and readability. Favor left-alignment.
- Employ subtle transitions and animations to enhance user experience without overwhelming the user.
